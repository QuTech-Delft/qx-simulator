Welcome to QX-simulator's documentation!
==================================

QX-simulator is a simulator for `cQasm <https://libqasm.readthedocs.io/en/latest/index.html>`_ quantum circuits.

It is developed at `QuTech <https://qutech.nl/>`_ in Delft, The Netherlands.

It comes both as an executable binary built from C++ and a Python wrapper API called QXelarator.
